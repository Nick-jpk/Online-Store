LINE STORE WITH BACKGROUND MUSIC
================================

HOW TO SET UP:
1. Download all files into one folder
2. Place your music file in the same folder
3. Rename your music file to: background-music.mp3
4. Open index.html in your web browser

FILE STRUCTURE:
- index.html      (Main website)
- style.css       (Styles)
- script.js       (JavaScript functionality)
- background-music.mp3 (Your background music - ADD THIS FILE)

REQUIREMENTS:
1. Music file must be MP3 format
2. File should be named "background-music.mp3"
3. Place all files in the same folder

FEATURES:
✅ Background music with auto-play
✅ Music player controls (play/pause, volume, mute)
✅ Visual music equalizer
✅ Responsive design
✅ Shopping cart simulation
✅ KES currency display
✅ Mobile-friendly

TO UPLOAD TO VERCEL:
1. Zip the entire folder
2. Go to vercel.com
3. Drag and drop the zip file
4. Your site will be deployed automatically

TROUBLESHOOTING:
- If music doesn't play, click the play button
- Ensure music file is in correct format (MP3)
- Check browser console for errors (F12)
- Make sure all files are in same folder

ENJOY YOUR MUSIC-ENABLED STORE!
